• Sirve para pedir datos.

  • JavaScript Tutor no permite programas que pidan datos.

  • El ejemplo que aparece después utiliza el sitio. https://gilpgnod.github.io/, que permite ejecutar y depurar programas que utilicen question y console.log.

  • El ambiente Node.js, que sirve para desarrollar aplicaciones de consola, servidores web y programar dispositivos para IoT, puede usar la función question, pero primero tienes que importarla de la librería readline-sync.

Ejemplo

const nombre = question("Nombre: ")
console.log("Hola", nombre)
